# Chess AI ♟️

An AI-powered chess game built with Next.js and TypeScript.

## 🔗 Live Demo
https://chess-ai-smoky.vercel.app/ 

## 🛠 Tech Stack
- Next.js
- TypeScript
- Minimax Algorithm with Alpha-Beta Pruning

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com/Harshshah931/chess-ai.git`
2. `cd chess-ai`
3. `npm install`
4. `npm run dev`
5. Open `localhost:4028`
